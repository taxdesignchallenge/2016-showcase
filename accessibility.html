Accessibility Guide

This guide is intended to assist individuals with disabilities in their use of the IRS Design Challenge website. The guide provides direction in accessing the features available using specific technologies, including screen readers, screen magnifiers and voice command systems.

Known accessibility issues: PDFs hosted on the website may not be accessible to Assistive Technology users. Please email for further information or to receive a submission in an accessible format.

To view more information on IRS.gov Accessibility, go to the IRS.gov Accessibility page.

Rotor

The Rotor allows the user to choose from a list of navigation options, including Containers, Headings, Landmarks, Same item, Vertical Navigation, Static text, Links, Tables, Lists, Buttons, Form Controls, Text fields, Search Fields, and Images.**

Twist two fingers in a circle to choose a setting on the rotor. Left and Right swipes go to the next element on the page, Up and Down swipes go the next element of type selected in the rotor.

Dragon Naturally Speaking Actions and Commands
Action Commands
Read entire page: Two finger swipe down
Read next item: Swipe right
Activate link, button: Double tap
Close popup, cancel last action: Two finger scrub
Go to next heading: Rotor, swipe down
Go to next landmark/region: Rotor, swipe down
Go to next table: Rotor, swipe down
Navigate table cells: Swipe up, down, left or right
Go to next list: Rotor, swipe down
Go to next list item: Swipe right
Go to next image: Rotor, swipe right
Go to next link: Rotor, swipe down
Go to first (last) item on page: 4 finger tap upper (lower) half of screen
Scroll (vertically or horizontally): 3 fingers swipe up, down, left, or right
Go back to previous heading, landmark, table, focusable item, etc.: Swipe up
